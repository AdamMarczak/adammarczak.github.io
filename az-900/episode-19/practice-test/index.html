Practice Test

Question 1

Contoso company wants to introduce Azure to their developers. Which is the best tool to perform simple tasks and learn Azure platform?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure Portal allows users to perform simple operations and manage their Azure environments. It was designed with self-service in mind, and as such it has a lot of navigation and guides making it perfect solution for new users.

Question 2

Azure CLI is ...

✔ That's correct
❌ That's not correct
👨‍🦱 💬 CLI stands for Command Line Interface and it’s a tool which integrates with existing command line terminals allowing users to manage their Azure resources.

Question 3

Azure Cloud Shell is ...

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure Cloud Shell is a small cloud-based environment with lots of preinstalled tools that allow users to manage their Azure resources.

Question 4

What are the interfaces that users can use to access Azure Cloud Shell?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 All of mentioned interfaces can be used to access Azure Cloud Shell.

Question 5

Azure PowerShell is ...

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure PowerShell is a name for PowerShell module that allows customers to perform common administrative tasks with their Azure resources.

Question 6

Contoso administrators use Mac-based machines in their work. Which tools can they use to manage Azure resources?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 All of the mentioned tools are either multi-platform or browser based, as such they can be used on any operating system.

Question 7

Which Azure tools are designed to help with common operational and automation tasks in Azure?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure CLI and PowerShell are designed to help with common operational and automation tasks. With use of Cloud Shell administrators and developers can perform their tasks directly in the cloud.

Question 8

Contoso company wants to move and automate their infrastructure in the cloud. Their current stack is based mostly on Windows Servers. What is the best tool for their administrators to learn at first when considering their existing skill-set?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Administrators working with Windows-based systems most likely already know PowerShell, as such Azure PowerShell would make the most sense for them.
