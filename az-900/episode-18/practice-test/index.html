Practice Test

Question 1

Which sentence best describes what DevOps is?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 DevOps is a very broad topic and can mean anything. But in general, it is a set of practices that help with project delivery. Azure DevOps is just one of many available tools for building solutions using DevOps.

Question 2

Contoso company wants to allow testers to quickly provision clean environments so that they have a sandbox environment for their testing scenarios and automation. Which service fits their need best?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure DevTest Labs is a workspace based service in Azure that allows customers for self-service provisioning of Azure Virtual Machines with extra tools (artifacts) and a lot of additional automation options.

Question 3

What is the name of the service for building ci/cd pipelines with private git repositories and features to track project work?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure DevOps is a collection of tools that allow for end-2-end development of applications and solutions by providing customers with a set of features like Boards (task/work management), Repos (git repositories), Pipelines (ci/cd pipelines), etc.

Question 4

Contoso administrators want to allow developers to test their software in the cloud by providing them with the ability to create virtual machines at any time. Being cautious of the cost and management effort, which service should administrators choose?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure DevTest Labs is the best choice here. This service will allow customers to provision machines using a concise workspace while providing administrators with a lot of control using built-in policies.

Question 5

Azure DevOps delivers multiple services for building solutions based on DevOps practices. Which services are those? Choose 5.

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Azure DevOps consists of multiple services (boards, pipelines, repos, test plans, and artifacts) to support the end-2-end development life-cycle for our applications. Azure marketplace; while not a core-service; allows to extend features of those services with a third-party provided additions.

Question 6

Contoso manager asks the team to implement continuous deployment processes for their application. The team decided to use Azure DevTest Labs. Does this service meet their needs?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 No. Azure DevTest labs is used to provide sandbox environments to developers and testers but it doesn’t deliver any features for building CI/CD pipelines.

Question 7

Contoso manager asks the team to implement continuous deployment processes for their application. The team decided to use Azure DevOps. Does this service meet their needs?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Yes. Azure DevOps comes with a pipelines feature that allow developers to create CI/CD pipelines for their applications easily.
