Practice Test

Question 1

Cloud computing is a delivery model for what four service types?

✔ That's correct
❌ That's not correct
👨‍🦱 💬 While cloud computing contains a number of service types, the four most common ones by the definition are Compute Power, Storage, Analytics, and Networking.

Question 2

Match words from the boxes with the numbers in the picture below.

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Increasing the size of the resource is called scaling up, hence decreasing the size is scaling down. Because we move along the vertical line, changing the size is called vertical scaling. Increasing the amount of instances of our resource is called scaling out, hence decreasing the amount is scaling in. Because we move along the horizontal line, changing the size is called horizontal scaling.

Question 3

Match the 3 most appropriate words that best describe how the cloud helps with the allocation of resources.

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Scalability is the system's ability to scale (allocate/deallocate) resources. Agility is the ability to scale quickly. If that scaling is happening automatically, then we are talking about Elasticity. All of those terms directly explain the system's ability to change resources at will.

Question 4

A systems ability to scale dynamically is called _________.

✔ That's correct
❌ That's not correct
👨‍🦱 💬 When the system is elastic that means it can adjust its processing power to match the demand. This happens only when the system is able to allocate and deallocate resources dynamically.

Question 5

Ability of the system to run for extended periods of time without any interruption is ...

✔ That's correct
❌ That's not correct
👨‍🦱 💬 Availability talks about system uptime, so when the system is available to users/other applications. High Availability simply means that system is designed to run for a very long period of time.
